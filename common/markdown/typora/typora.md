# 使用技巧

## 1、如何在任意位置自定义锚点

两种方式：

1. 跳转到某个级别的标题栏

   例如：

   ```markdown
   # titleA
   # titleB
   - [toA](#titleA)
   - [toB](#titleB)
   ```

2. 使用`<a>`定义锚点，完成跳转

   例如：

   ```markdown
   <a id="positionA">锚点显示的内容</a>
   - [toPositionA](#positionA)
   ```
